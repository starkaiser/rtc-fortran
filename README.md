## Build Instructions

### Linux

You will need:
* GNU Fortran
* Make

To build, simply run:

```
$ make
```
This will build and run the tests.

### Windows

You will need:
* Intel Fortran Compiler (ifx)
* Visual Studio 2026

The Windows VS solution is in /rtc-fortran.
